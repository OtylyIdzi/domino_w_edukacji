# Analiza wpływu środowiska rodzinnego na wyniki edukacyjne uczniów  
Projekt w R

## Opis projektu
Projekt analizuje zależności między środowiskiem rodzinnym, stylem życia a wynikami edukacyjnymi uczniów portugalskich szkół średnich. Badanie opiera się na danych z UCI Machine Learning Repository i koncentruje się na koncepcji „efektu domina” – czyli sekwencji zależności, w której czynniki rodzinne i społeczne wpływają na zachowania ucznia, jego zaangażowanie oraz końcowe oceny.

Celem analizy jest zrozumienie, w jaki sposób takie elementy jak relacje rodzinne, czas wolny, spożycie alkoholu, absencje czy czas nauki wpływają na strukturę osiągnięć szkolnych oraz wynik końcowy (G3).

## Wykorzystane dane
Dane pochodzą z dwóch zestawów:
- **student-mat.csv** – uczniowie uczestniczący w kursie matematyki  
- **student-por.csv** – uczniowie uczący się języka portugalskiego  

**Źródło danych:**  
P. Cortez and A. Silva (2008), *UCI Machine Learning Repository – Student Alcohol Consumption*.
